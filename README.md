## Правила загрузки лабораторных работ в репозиторий

<ol>
    <li>
       Лабораторные работы необходимо загружать в свою папку (необходимо будет создать личную папку, название = ваша фамилия)
    </li>
    <li>
       Название файла = номер лабораторной работы по списку
    </li>
    <li>
        Для каждой лабораторной работы назначен дедлайн, который оглашается на паре
    </li>
    <li>
        При загрузке лабораторных необходимо будет форкнуть репозиторий и создавать PR в ветку main, чтобы ваши изменения отобразились в общем репозитории
    </li>
</ol>
